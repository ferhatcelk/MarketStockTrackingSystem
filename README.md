# 🛒 Market Stok Takip Sistemi

Windows Forms kullanılarak geliştirilen Market Stok Takip Sistemi, market ürünlerinin kayıt altına alınmasını, stok bilgilerinin yönetilmesini ve temel envanter işlemlerinin gerçekleştirilmesini sağlayan masaüstü uygulamasıdır.

Bu proje, Bursa Uludağ Üniversitesi kapsamında verilen Windows Forms proje ödevi için geliştirilmektedir.

---

##  Kullanılan Teknolojiler

- C#
- .NET Windows Forms
- Visual Studio 2022
- Object-Oriented Programming (OOP)
- Git & GitHub

---

## 📌 Proje Özellikleri

- Ürün Ekleme
- Ürün Güncelleme
- Ürün Silme
- Ürün Listeleme
- Ürün Arama
- Kategoriye Göre Filtreleme
- Stok Hareketlerini Görüntüleme
- Kategori Yönetimi
- Veri Doğrulama
- Mouse Event Kullanımı

##  Proje Amacı

Marketlerde bulunan ürünlerin kayıt altına alınmasını, stok bilgilerinin düzenli bir şekilde takip edilmesini ve ürün yönetim işlemlerinin kolaylaştırılmasını sağlayan kullanıcı dostu bir masaüstü uygulaması geliştirmektir.
